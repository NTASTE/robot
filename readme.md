机器人自动聊天项目，采用 [图灵](http://www.tuling123.com/) 官方的Api接口。

## 使用扩展

   - [laravel-ip](https://github.com/jormin/laravel-ip)：解析IP地址。
   
   - [laravel-tuling](https://github.com/jormin/laravel-tuling)：集成[图灵](http://www.tuling123.com/) 官方的Api接口。
